Stock Market Dashboard Web App

This project display the stocks of various Tech Companies by fetching the data from the dataset of last 1 year.

This is the first version of my project. It will be getting many updates with time. To get the latest updated of the project follow my github.

This project is a perfect example of how we can use pandas and streamlit together for a working implementation.

It has a simple user interface so any one can run it very easily and check the stock of his/her favorite tech company.

This is a beginner friendly Python project which is very useful.

Technologies:
  • - Python
  • - Pandas
  • - Streamlit
  • - Heroku
