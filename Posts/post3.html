British Bakery

British Bakery is a dummy website for a bakery store. It is a web development project to get a good understanding of html , css and javascript. It uses beautiful color themes and animations to attract the audience.

The website is fully responsive can be view perfectly in any devices(mobiles, laptops , tablets etc).

It is an Open Source Project so if any one want to use this template can use it for free.
